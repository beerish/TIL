### 1. element과 container

element(요소) : 낱개 하나 하나
container : element를 담는 박스나 포장용기 같은 것. 

✨container는 semantic tag랑 유사한 것 같다. 컨테이너가 더 큰 범위인건 안다.


### 2. block과 inline
